---

SkillSpell is a web platform for building high-quality AI agent skills — structured
instruction sets that make Claude Code, Cursor, Windsurf, GitHub Copilot, and other
AI coding tools better at specific tasks. Describe what you need in plain English, and
SkillSpell generates, refines, evaluates, and distributes the skill for you.
