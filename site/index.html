$ and when you're ready to distribute

One command from spell to shipped

Publish from the web app, install from the terminal. The @skillspell/cli puts every approved skill one command away from any AI coding tool.

$ npm install -g @skillspell/cli
$ skillspell login --sso
$ skillspell install api-design-review -t claude
✓ Installed to .claude/skills/api-design-review
